# Pollution Palooza
A hackathon project created for MakueUC2022. Devpost found [here](https://devpost.com/software/pollution-palooza)

### Runing the Game 

1. Clone the repo 
```
git clone https://github.com/AdamWebbUCIS/PollutionPaloozaMakueUC2022.git
```

2. Install pygame 
```
pip install pygame
```

3. Run using main.py 
```
python3 main.py
```
